# dotfiles

Personal config files managed with [GNU Stow](https://www.gnu.org/software/stow/).

## Install

```bash
git clone git@github.com:ya-nsh/dotfiles.git ~/dotfiles
cd ~/dotfiles
stow zsh git nvim
```

## Structure

| Package | Config |
|---------|--------|
| `zsh`   | `~/.zshrc` |
| `git`   | `~/.gitconfig` |
| `nvim`  | `~/.config/nvim/` |
